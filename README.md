# Contrarian vs Conformist Debate Using AutoGen

Using AutoGen to develop two personalities (Contrarian and Conformist) to debate each other!

Create a OAI_CONFIG_LIST file in this format to run!

```
[
    {
        "model": "gpt-4",
        "api_key": ""
    },
    {
        "model": "gpt-3.5-turbo",
        "api_key": ""
    }
]
```
